
  # VoiceSewa 

  Multilingual Voice-Assisted Job Connection Platform for Blue-Collar Services

```
voicesewa_translate/
│
├── pubspec.yaml
├── firebase.json
│
├── firestore/
│   ├── firestore.rules
│   └── firestore.indexes.json
│
├── functions/
│   ├── index.js
│   └── package.json
│
├── android/app/src/main/
│   └── AndroidManifest.xml          ← full replacement file
│
├── ios/Runner/
│   └── Info_plist_additions.xml     ← merge-in instructions
│
└── lib/
    ├── main.dart                    ← app entry, Firebase + FCM init
    ├── firebase_options.dart        ← generated by flutterfire configure
    │
    ├── core/
    │   ├── constants.dart           ← backend URL, Firestore collection names
    │   ├── extensions.dart          ← duration formatter, BCP-47 helpers
    │   └── theme.dart               ← colours, text styles, animation durations
    │
    └── features/
        └── translate_call/
            │
            ├── data/
            │   ├── translate_call_repository.dart   ← REST + WebSocket + Firestore signalling
            │   ├── call_history_repository.dart     ← Firestore call log CRUD
            │   ├── fcm_service.dart                 ← FCM token management + message handling
            │   └── notification_service.dart        ← flutter_local_notifications, heads-up UI
            │
            ├── domain/
            │   ├── call_language.dart               ← enum, 10 Indian languages + BCP-47 codes
            │   ├── call_session.dart                ← Freezed model, Firestore serialisation
            │   └── call_history_entry.dart          ← Freezed model, per-user call log entry
            │
            ├── application/
            │   ├── providers.dart                   ← every Riverpod provider in one file
            │   ├── call_controller.dart             ← AsyncNotifier, full call lifecycle
            │   └── call_history_controller.dart     ← AsyncNotifier, call log stream
            │
            └── presentation/
                ├── language_picker_screen.dart      ← pick your language before/during call
                ├── outgoing_call_screen.dart        ← ringing, waiting for answer
                ├── incoming_call_screen.dart        ← accept / decline
                ├── active_call_screen.dart          ← live call, captions, mute, end
                ├── call_history_screen.dart         ← call log with swipe-to-delete
                └── home_screen.dart                 ← contact list + initiate call
```

```
dart run build_runner build --delete-conflicting-outputs
```
